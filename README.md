# 《下沉》

> 一条河。一个渡口。十年。

方远在规划局坐了六年，辞职了。他去了一个废弃的渡口，买了一条旧船，开始做货运。

没有人觉得他能做成。包括他自己。

三年后，他有了自己的码头。

这不是一个关于逆袭的故事。这是一个关于等待的故事——等水涨，等船来，等一个人想明白。

## 关于本书

**字数**：约 9 万字

**标签**：现实主义 / 长篇小说 / 中国当代文学 / 水运题材

**核心意象**：

> “不要追水。等水流到你这里。”

**人物**：
- **方远**：从规划局辞职，在废弃渡口重新开始
- **鲁建国**：被体制抛弃的中年人，方远唯一的合伙人
- **陈敏**：面馆老板娘，沉默的见证者
- **老赵**：化工厂老厂长，坚守了三十多年
- **林老板**：地产商，最后成了方远最大的客户
